# Data Availability Statement

## 현재 제출본용 문구 (확정)

The data that support the findings of this study are available in a public GitHub repository (`https://github.com/mulgae-life/ib-lbm-comparison`). Processed benchmark data, analysis scripts, simulation configuration files, and source code are provided. Additional raw simulation outputs are available from the corresponding author upon reasonable request.

## Zenodo DOI 확보 후 최종 문구

The data that support the findings of this study are openly available in GitHub at `https://github.com/mulgae-life/ib-lbm-comparison` and are archived in Zenodo at `[DOI]`. Processed benchmark data, analysis scripts, simulation configuration files, and source code are provided. Additional raw simulation outputs are available from the corresponding author upon reasonable request.
